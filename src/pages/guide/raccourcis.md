# Raccourcis clavier

## Raccourcis globaux

Ces raccourcis fonctionnent même quand Engra est en arrière-plan :

| Raccourci | Action |
|-----------|--------|
| Cmd+Shift+R | Démarrer/arrêter l'enregistrement |
| Cmd+Shift+N | Note vocale rapide |

## Dans l'application

| Raccourci | Action |
|-----------|--------|
| Cmd+K | Recherche rapide |
| Cmd+N | Nouvelle note |
| Cmd+, | Paramètres |

## Personnalisation

Les raccourcis globaux sont configurables dans **Paramètres > Raccourcis**.

> **Note Linux/Windows** : Remplacez `Cmd` par `Ctrl`.
