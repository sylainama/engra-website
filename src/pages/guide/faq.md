# FAQ

## Général

**Engra est-il vraiment 100% local ?**

Oui. Les seules connexions réseau sont :
- La vérification de licence — une fois par mois
- Le téléchargement initial des modèles IA
- Les imports YouTube (optionnel)

Aucun audio, transcription ou résumé n'est jamais envoyé à un serveur.

**Faut-il une connexion internet ?**

Non, après l'installation initiale. Engra dispose d'une période de grâce de 30 jours pour la validation de licence hors-ligne.

**Quelles langues sont supportées ?**

Le moteur de transcription supporte 90+ langues. L'interface d'Engra est en français.

## Technique

**Quelle quantité de RAM faut-il ?**

- **8 Go** : suffisant pour la transcription + les modèles IA légers
- **16 Go** : recommandé pour des modèles plus avancés

**Combien d'espace disque ?**

- Engra : ~200 Mo
- Modèle de transcription : ~500 Mo
- Modèle IA pour résumés : 1-8 Go selon le niveau choisi
- Enregistrements : ~5 Mo/minute (WAV)

**Puis-je utiliser mes propres modèles IA ?**

Oui. Tout modèle compatible installé sur votre machine est utilisable dans Engra.

## Licence

**Puis-je transférer ma licence ?**

Oui. Désactivez un appareil dans **Paramètres > Licence**, puis activez sur le nouveau.

**Politique de remboursement ?**

30 jours satisfait ou remboursé, sans questions.
