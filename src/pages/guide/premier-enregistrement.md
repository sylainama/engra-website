# Premier enregistrement

## Lancer un enregistrement

1. Ouvrez Engra
2. Cliquez sur le bouton **Enregistrer** ou utilisez le raccourci **Cmd+Shift+R** (macOS)
3. Choisissez les sources audio :
   - **Microphone** : votre voix
   - **Audio système** : le son des autres participants (Zoom, Teams, etc.)
4. Cliquez sur **Démarrer**

## Détection automatique

Engra détecte automatiquement les réunions en cours :
- **Zoom**, **Teams**, **Google Meet**, **Slack Huddle**, **Discord**
- Une bannière apparaît pour vous proposer d'enregistrer

Vous pouvez activer l'**auto-record** dans les paramètres pour que l'enregistrement démarre automatiquement.

## Pendant l'enregistrement

- La transcription se fait **en temps réel** (toutes les 5 secondes)
- Un résumé en direct est généré si l'IA est active
- L'icône dans le menu bar passe en rouge

## Arrêter l'enregistrement

- Cliquez sur **Stop** ou utilisez **Cmd+Shift+R**
- Engra lance automatiquement :
  1. La transcription complète
  2. Le résumé IA
  3. L'extraction des entités pour le graphe

## Templates de réunion

Choisissez un template adapté pour de meilleurs résumés :
- **Général** : résumé standard
- **Standup** : focus sur blocages et avancements
- **Rétro** : ce qui a marché, à améliorer, actions
- **Sales call** : besoins client, objections, prochaines étapes
- **Interview** : compétences, impressions, décision
- **Brainstorm** : idées générées, votes, sélection
