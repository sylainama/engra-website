# Transcription

## Fonctionnement

Engra utilise un moteur de transcription IA optimisé qui tourne intégralement sur votre machine. Aucune donnée audio n'est envoyée à l'extérieur.

## Modes de transcription

### Temps réel
Pendant l'enregistrement, Engra transcrit **par segments de 5 secondes**. Vous voyez le texte apparaître en direct.

### Post-enregistrement
Après l'arrêt, une transcription complète est générée sur l'audio final (meilleure qualité).

## Diarisation

Engra identifie les différents locuteurs dans la transcription. Les locuteurs sont affichés comme **[Locuteur 1]**, **[Locuteur 2]**, etc.

## Langues supportées

Le moteur supporte plus de 90 langues. Les plus courantes :
- Français, Anglais, Espagnol, Allemand, Italien, Portugais
- Japonais, Chinois, Coréen, Arabe

La langue est détectée automatiquement ou configurable dans les paramètres.

## Choix du modèle

Plus le modèle est gros, meilleure est la qualité, mais plus il faut de ressources. Le modèle **Standard** offre le meilleur compromis pour la plupart des utilisateurs.
