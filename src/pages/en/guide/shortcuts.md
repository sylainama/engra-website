# Keyboard shortcuts

## Global shortcuts

These shortcuts work even when Engra is in the background:

| Shortcut | Action |
|----------|--------|
| Cmd+Shift+R | Start/stop recording |
| Cmd+Shift+N | Quick voice note |

## In the application

| Shortcut | Action |
|----------|--------|
| Cmd+K | Quick search |
| Cmd+N | New note |
| Cmd+, | Settings |

## Customization

Global shortcuts can be configured in **Settings > Shortcuts**.

> **Linux/Windows note**: Replace `Cmd` with `Ctrl`.
